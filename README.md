# mrqsmar.github.io

Personal portfolio website for **Marques Mar** — Technical Product Manager specializing in ML-powered payments and AI products at Visa.

## Live Site

[https://mrqsmar.github.io](https://mrqsmar.github.io)

## Tech Stack

- **React 19** with **TypeScript** — component-based UI
- **Vite 6** — fast build tooling and HMR
- **Tailwind CSS 3** — utility-first styling
- **Framer Motion** — scroll-reveal and entrance animations
- Google Fonts (Inter)
- Fully mobile responsive

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Deploy

Pushes to `main` trigger a GitHub Actions workflow that builds and deploys to GitHub Pages automatically.
